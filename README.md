# Звіт з лабораторної роботи №5
## Моніторинг та алертинг енергетичних систем

## Варіант 1: Сонячні електростанції Київської області
### Підваріант А: Моніторинг продуктивності в реальному часі

## Мета роботи

Побудувати комплексну систему моніторингу для сонячного парку, що складається з 50 сонячних електростанцій, з використанням Prometheus та Grafana. Отримати практичні навички збору метрик у реальному часі, симуляції технологічних процесів (сонячний цикл, хмарність) та налаштування автоматичного сповіщення про критичні стани обладнання.

## Вихідні дані варіанта

Об’єктом дослідження є 50 сонячних електростанцій Київської області. Для моніторингу використовуються такі метрики: потужність генерації (МВт), температура панелей (°C), сонячна радіація (Вт/м²), коефіцієнт корисної дії (%). Симуляція роботи станцій виконується лише в межах світлового дня з 06:00 до 20:00 з урахуванням коефіцієнта хмарності в діапазоні 0.5–1.0.

## Хід роботи

### Завдання 1. Налаштування Prometheus

Було встановлено Prometheus Server та створено конфігураційний файл `prometheus.yml` для збору метрик з Python-експортера, що працює на порту 8000.

### Завдання 2. Створення Python-експортера

Розроблено скрипт `solar_exporter.py` з використанням бібліотеки `prometheus_client`. Скрипт симулює роботу 50 сонячних електростанцій з урахуванням сонячної активності залежно від поточної години комп’ютера.


### Завдання 3. Створення Dashboard у Grafana

Prometheus було підключено як джерело даних (Data Source) у Grafana. Створено інтерактивний дашборд з такими панелями:

- Power Output (Time Series) — динаміка генерації станцій
- Panel Temperature (Gauge) — поточний тепловий стан із порогами (зелений < 50°C, жовтий 50–60°C, червоний > 60°C)
- Solar Radiation (Time Series) — інтенсивність сонячного випромінювання
- Efficiency Performers (Bar Chart) — топ-5 найефективніших станцій



### Завдання 4. Налаштування алертингу

Створено файл `alerts.yml` з правилами для виявлення перегріву (>60°C), низької ефективності (<12%) та відсутності генерації вдень.
